### About
An app can makes survey, send to many recipients (text/ html/ attachments). Tracking response. Notify realtime on slack chanel

### Clone
 - git clone https://github.com/blackblue2712/send-surveys.git

### Setup
    - cd to project folder
    - npm install
    - cd ./client
    - npm install
    - cd -
    - npm run dev
